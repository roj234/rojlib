// 同时字符串将通过Tokenizer进行预处理，替换并分割未转义的{}，并根据括号配对分割语句
// 请注意：if () 或下列任何语句不加花括号都是不合法的，会直接报错
// 目前允许的语句：if for while do switch with @
// 如果需要更多的灵活性，请使用 {{ code block }} 直接插入代码 (在代码中你可以定义变量，写入内容需要使用隐含在方法参数中的`response.append(*)`)
// 隐含变量是自动prepend的方法签名，包括(final Request request, final CharList response)
// 其他hint包括request.responseHeader().put、request.context().get等
// 默认情况下，通过{}输出的字符串会自动进行HTML转义，如果不想这么做，需要使用 {@raw( ... )}
